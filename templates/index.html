Live Endpoints

  • GET/live/session/currentCurrent session cached in memory
  • GET/live/timing/snapshotComplete live snapshot
  • GET/live/timing/streamSSE stream with update/heartbeat events
  • POST/live/reloadForces one immediate poll cycle

Legacy Catalog Flow

  • GET/legacy/catalog/yearsReturns available years for FastF1 schedule data
  • GET/legacy/catalog/events?year=2024Returns events and sessions available for the selected year
  • GET/legacy/catalog/drivers?year=2024&trackName=Monaco&session=QReturns drivers with available telemetry for the selected session
  • GET/get-telemetry?year=...&trackName=...&session=...&driverName=...Generate telemetry PDF using values selected from catalog endpoints
  • GET/get-telemetry-compare?year=...&trackName=...&session=...&driverA=...&driverB=...Generate comparison PDF with overlaid charts for two selected drivers

Legacy Endpoints

  • GET/statusCompact backend health/status
  • GET/get-telemetryLegacy telemetry PDF endpoint (required params: year, trackName, session, driverName)
  • GET/get-telemetry-compareComparison PDF for two drivers (required params: year, trackName, session, driverA, driverB)

Per-Driver Data

  • Position, gap-to-leader, interval
  • Lap time, sectors, microsectors
  • Tyre compound and laps on current tyre
  • Pit status and driver/team metadata

Microsector Labels

  • 0 -> no_data
  • 2048 -> slower
  • 2049 -> improved
  • 2051 -> best_overall
  • 2064 -> pitlane
  • altri -> unknown

Telemetry Compare API Guide

  1. Call /legacy/catalog/years and select the season.
  2. Call /legacy/catalog/events?year=... and pick event_name + session code.
  3. Call /legacy/catalog/drivers?year=...&trackName=...&session=... and pick two drivers.
  4. Download compare PDF with /get-telemetry-compare?year=...&trackName=...&session=...&driverA=...&driverB=....
curl -v "http://localhost:5050/get-telemetry-compare?year=2024&trackName=Monaco&session=Q&driverA=VER&driverB=LEC" -o compare.pdf
200 OK
content-type: application/pdf

400 (missing params)
{"error":"Missing required parameters: year, trackName, session, driverA, driverB"}

Compare PDF overlays speed/throttle/brake for two drivers. Corner numbers and corner-speed labels are shown only when FastF1 corner metadata is available.
